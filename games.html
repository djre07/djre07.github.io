This is a game I called COLOR JUMP. The general story of the game is about a cube who lost its colors and the player controls it with the goal to get the colors back. There are three basic levels, each increasing in difficulty. I created the basic mechanics and pixel art for the game. You can play it here!

This was a practice game I created just for fun using Godot. After learning basic skills for 3D modeling, I made a hand model which then I got the idea to create a first person game for my first offical 3D game. In this game you punch basic waterbottle enemies (seen in picture), in a small neighborhood that is surrounded by mountains. I created all assets in the game and coded everything as well.
